## 📝 Writeup Format

Each writeup typically includes:

* Challenge description
* Official hints (if available)
* Initial analysis
* Exploitation steps
* Flag
* Notes / lessons learned

All writeups are written in Markdown (`.md`) for readability and ease of use on GitHub.
